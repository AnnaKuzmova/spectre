Конфигурация

Навигирайте настоящата си директория в папка myapp, отворете bash terminal и изпълнете следните команди:
 - npm install - инсталира dependency-тата на проекта в папка node_modules
 - npm run json-server - стартира мокнат локален уеб сървър на порт 8000 (при зает порт моля освободете го)
 - npm run start - стартира приложението на порт 3000

Важно!
Не изтривайте или модифицирайте папка db.json, в нея се пазят данните обработвани чрез приложението.
